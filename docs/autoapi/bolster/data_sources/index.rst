bolster.data_sources
====================

.. py:module:: bolster.data_sources

.. autoapi-nested-parse::

   Data sources - mostly Northern Ireland focused stuff.

   Various scrapers and API wrappers for NI government data:
   - ni_water: Water quality by postcode/zone
   - metoffice: UK precipitation maps
   - wikipedia: NI Executive history parsing
   - ni_house_price_index: Housing market data
   - cineworld: Movie listings
   - companies_house: UK company data
   - eoni: Electoral data

   Most have corresponding CLI commands.
