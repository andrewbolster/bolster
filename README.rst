Features
--------

* Efficient tree/node traversal and iteration
* Datetime helpers
* Concurrecy Helpers
* Web safe Encapsulation/Decapsulation helpers
* `pandas`-esque `aggregate`/`transform_r` functions
* "Best Practice" AWS service handling
